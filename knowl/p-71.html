A decription for the RELAX-NG schema can be found at http://mathbook.pugetsound.edu/doc/schema/. For instance, if you select the element <example> in the left-hand column, you will see lists of elements under the headings “Content model elements” and “Included in content model of elements.” The former list contains the elements that the <example> element may contain, and the latter contains the elements in which the <example> element may be contained.
