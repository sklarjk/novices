  • From the directory containing your PTX file, type in the same command as you used to create the HTML document, replacing the mathbook-html.xsl with mathbook-latex.xsl. That is, enter ~/xsltproc/xsltproc --xinclude ~/mathbook/xsl/mathbook-latex.xsl hello-world.xml. This should create a LaTeX file hello-world.latex in the same hello-world folder.
